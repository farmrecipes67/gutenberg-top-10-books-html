## 🚀 Usage

Since all books are provided as standalone HTML files, no installation or build step is required.

### View in a Browser

Simply open any of the HTML files directly in your web browser:

```bash
# Clone the repository
git clone https://github.com/farmrecipes67/gutenberg-top-10-books-html.git

# Navigate into the books directory
cd gutenberg-top-10-books-html/books

# Open a book (example for macOS)
open pride-and-prejudice.html

# On Linux
xdg-open pride-and-prejudice.html

# On Windows
start pride-and-prejudice.html
```
